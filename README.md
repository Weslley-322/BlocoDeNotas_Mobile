# 📝 Bloco de Notas Mobile

Este projeto é uma aplicação de gerenciamento de notas desenvolvida com **React Native** e **Expo**. O que nasceu como uma atividade acadêmica evoluiu para uma ferramenta funcional que permite a criação, organização e persistência de anotações de forma simples, intuitiva e multiplataforma.

## 🚀 Tecnologias e Funcionalidades

A aplicação utiliza as versões mais recentes do ecossistema mobile para garantir performance e uma experiência de usuário (UX) fluida:

* **Core & Navegação:** Desenvolvido com **React 19** e **Expo**, utilizando o **React Navigation (Native Stack)** para garantir transições nativas entre a listagem e os detalhes das notas.
* **Gestão de Dados:** Integração com **Async Storage** para persistência local e **Axios** para consumo de dados, garantindo que suas anotações estejam sempre disponíveis.
* **Interface:** Design focado em dispositivos móveis com suporte a **Safe Area Context**, garantindo que o app se adapte perfeitamente a diferentes entalhes (notches) de telas Android e iOS.

## 🛠️ Instalação e Execução

Para rodar o projeto localmente, siga os passos abaixo:

1.  **Clone e Acesse:**
    ```bash
    git clone [https://github.com/Weslley-322/BlocoDeNotas_Mobile.git](https://github.com/Weslley-322/BlocoDeNotas_Mobile.git)
    cd BlocoDeNotas_Mobile
    ```

2.  **Instale as Dependências:**
    ```bash
    npm install
    ```

3.  **Inicie o Expo:**
    ```bash
    npx expo start
    ```

4.  **Visualize o App:**
    Abra o app **Expo Go** no seu celular e escaneie o QR Code gerado no terminal, ou pressione `a` para Android e `i` para iOS caso possua um emulador configurado.
